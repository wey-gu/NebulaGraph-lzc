# Build

## Build Portal

```bash
cd nebulagraph-lazycat
npm i
npx next build
cd ..
mkdir -p public
cp -r nebulagraph-lazycat/out/* public/
docker build -t weygu/nebulagraph-lzc-portal:v0.1.0 .
```

## Prepare `manifest.yml` and primitive image

```bash
lzc-dtl -c docker-compose.yaml
```
